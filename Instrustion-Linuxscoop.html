01. Initial Setup

Update your system

Ubuntu/Debian

sudo apt update && sudo apt dist-upgrade -y

Arch Linux - Manjaro

sudo pacman -Syyu

Fedora

sudo dnf upgrade

openSUSE

sudo zypper refresh && sudo zypper update

Disable Desktop Icon

System settings > Desktop > Desktop Layout > No Desktop icon

Window Center

System settings > Windows > Behavior > Location of newly ‚Ä¶ select Center

Install Nautilus and Set it as Default File Manager

Ubuntu/Debian

sudo apt install nautilus nautilus-admin nautilus-extension-gnome-terminal

Arch Linux - Manjaro

sudo pacman -Sy nautilus nautilus-admin nautilus-extension-gnome-terminal
pamac build nautilus-admin nautilus-open-any-terminal

Fedora

sudo dnf install nautilus gnome-terminal-nautilus

openSUSE

sudo zypper install nautilus gnome-terminal-nautilus

Set Nautilus as the default file manager

System Settings > Preferences > Preferred Applications

File manager: Files (nautilus)

03. Install GTK Theme

Extract and Install theme.zip to the home folder. From the terminal, run the following command:

unzip ~/Downloads/themes.zip -d ~/

Change folder ~/themes to ~/.themes

mv ~/themes/ ~/.themes

04. Install Icon Theme and start-menu-icon

Extract and Install icons.zip to $HOME/.local/share/ directory

unzip ~/Downloads/icons.zip -d ~/.local/share/

Extract and Installstart-menu-icon.zip to $HOME/.local/share/ directory

unzip ~/Downloads/start-menu-icon.zip -d ~/.local/share/icons/

05. Install Cursors Theme

Extract and Install cursors.zip to $HOME/ directory

unzip ~/Downloads/cursors.zip -d ~/

Change folder ~/cursors/ to ~/.icons

mv -v ~/cursors/ ~/.icons

06. Install Fonts and Wallpapers

Extract and Install fonts.zip to $HOME/.local/share/ directory

unzip ~/Downloads/fonts.zip -d ~/.local/share/

Extract and Install wallpapers.zip to /usr/share/backgrounds/ directory

sudo unzip ~/Downloads/wallpapers.zip -d /usr/share/backgrounds/

07. Install Cinnamon Applets and Config

Extract and Install cinnamon-applet-config.zip to $HOME directory

unzip ~/Downloads/cinnamon-applet-config.zip -d ~/Downloads/
rsync -av ~/Downloads/cinnamon-applet-config/.cinnamon ~/
rsync -av ~/Downloads/cinnamon-applet-config/.local ~/
rm -rfv ~/Downloads/cinnamon-applet-config

08. Apply Theme, Icon, Cursors Fonts, and Restore Cinnamon Config

Restore Cinnamon Config

Open terminal,

Extract cinnamon-config.zip and enter to folder cinnamon-config

unzip ~/Downloads/cinnamon-config.zip -d ~/Downloads/

Restore the cinnamon config with the following command

dconf load /org/cinnamon/ < ~/Downloads/cinnamon-config/cinnamon-catppcn.conf

Change the default cinnamon menu

Right-click on the menu icon > select configure > Import from file, Browse to ~/Download/cinnamon-config select file cinnamon-menu-config.json

Change Fonts

System settings > Font Selections

09. Install and Config Plank Dock

Installing Plank-Dock in major Linux OS

Ubuntu/Debian

sudo apt install plank

Arch Linux/Manjaro

sudo pacman -S plank

Fedora

sudo dnf install plank

openSUSE

sudo zypper install plank

Installing plank-dock theme and config

Extract plank-theme-config.zip

unzip ~/Downloads/plank-theme-config.zip -d ~/Downloads/
rsync -av ~/Downloads/plank-theme-config/.config ~/
rsync -av ~/Downloads/plank-theme-config/.local ~/
rm -rf ~/Downloads/plank-theme-config

Run plank. From the start menu type plank . launch plank preferences by hitting the button Ctrl + right click on plank-dock, change theme to cattpcn-dark

11. Installing Conky and Conky Config

Installing Conky in Major Linux OS

Ubuntu/Linux Mint/Debian

sudo apt install conky-all jq curl moc

Arch Linux/Manjaro

sudo pamac build conky-lua jq curl moc

Fedora

sudo dnf install conky jq curl moc

OpenSUSE

sudo zypper install conky jq curl moc

Open and Change file /etc/environment

sudo nano /etc/environment

add the line :

MUFFIN_NO_SHADOWS=1

Install conky config

  • Extract conky-config.zip.
unzip ~/Downloads/conky-config.zip -d ~/Downloads/
  • Move folder conky to ~/.config/
rsync -av ~/Downloads/conky-config/conky ~/.config
  • Move folder autostart to ~/.config/
rsync -av ~/Downloads/conky-config/autostart ~/.config
rm -rf ~/Downloads/conky-config

12. Install and Config Glava Audio Visualizer

Installing Glava audio visualizer on Linux

Ubuntu/Linux Mint/Debian

Install Dependency:

sudo apt-get install libgl1-mesa-dev libpulse0 libpulse-dev libxext6 libxext-dev libxrender-dev libxcomposite-dev liblua5.3-dev liblua5.3 lua-lgi lua-filesystem libobs0 libobs-dev meson build-essential gcc
sudo ldconfig
cd Downloads/
git clone https://gitlab.com/wild-turtles-publicly-release/glava/glava.git
cd glava
meson build --prefix /usr
ninja -C build
sudo ninja -C build install
glava --copy-config

Arch Linux/Manjaro Linux/EndeavourOS

sudo pacman -S glava
glava --copy-config

Fedora

sudo dnf -y install glava
glava --copy-config

OpenSUSE

sudo zypper install glava
glava --copy-config

Config GLAVA

Extract glava-config.zip, Copy/Move file .config to $HOME folder 

unzip ~/Downloads/glava-config.zip -d ~/Downloads/
rsync -av ~/Downloads/glava-config/.config ~/
rm -rfv ~/Downloads/glava-config

13. Additional Settings

Install profile Catppuccin for GNOME terminal

Make sure the dconf and python3 commands are available on your system.

Execute the ./install.py script, either after cloning the repository, or via curl:

curl -L https://raw.githubusercontent.com/catppuccin/gnome-terminal/v0.2.0/install.py | python3 -

In Gnome Terminal, open Edit -> Preferences, and enable the profile for the theme you want.

Custom Firefox Web Browser

Open terminal

cd ~/Downloads
git clone https://github.com/vinceliuice/WhiteSur-firefox-theme.git
cd WhiteSur-firefox-theme
./install.sh -m

Revert Back to the default settings

Open the terminal, and type the following command:

dconf reset -f /org/cinnamon/
